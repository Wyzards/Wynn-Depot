# Wynn-Depot
A private image database to catalogue collected items, for the Minecraft MMORPG server "Wynncraft"

This website was designed for a friend's personal usage for catalouging and annotating images of collected items, for the Minecraft server, Wynncraft.

Here's a view of the user interface:
![wynn depot preview](https://github.com/Wyzards/Wynn-Depot/blob/main/wynn_depot_preview.png?raw=true)

It features a search bar for searching database item entries, filters for categorizing and sorting items, and the ability to add any new entries added to the public item api.
